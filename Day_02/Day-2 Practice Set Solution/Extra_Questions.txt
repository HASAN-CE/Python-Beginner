### Variables

2. Write a Python script to calculate the square and cube of a number using variables.
3. Create a program that demonstrates the use of global and local variables.
4. Write a Python program to check the type of a variable using the `type()` function.
5. Assign multiple values to multiple variables in a single line and print them.
6. Write a program that shows variable re-assignment with different data types.
7. Create a program that calculates the compound interest using variables.
